자바 스크립트에서는 크게 number, string이라는 자료형을 가진다.
정수와 실수를 따로 나누지 않고 number로 표기하며
문자열은 당연히 string이다.

1. 변수 선언과 초기화

var a = 1;
a에 1이 저장된다.

var str = "최천일";
str에 최천일 이 저장된다.

각각의 자료형은 초기화되면서 결정된다.

2. 변수 타입 확인하기
typeof() 함수를 사용한다.

3. 문자열로 저장된 숫자를 타입 변환하기

만약에 프롬프트를 통해 숫자를 입력 받았을 경우에
var num = prompt("숫자를 입력해주세요");
prompt()는 string을 반환하기 때문에 num에 저장된 값의 타입은 string이다.

이것을 parseInt, parseFloat 함수를 사용하여 number타입으로 변환이 가능하다.
var a = "1.5";
var a_int = parseInt(a); // 정수만 추출
var a_float = parseFloat(a); // 실수 그대로 저장

예를 들어 문자열에 "길이 1cm"가 parseInt()를 사용한다면 NaN이 발생한다.
Not a Number의 뜻으로 숫자가 아니라는것이다.
하지만 "1cm"를 다시 parseInt()하면 1을 변수에 저장이 가능하다.
단지, 숫자가 가장 앞에 오면 그 숫자만 추출하는것이다.

4. 문자열 활용

자바 스크립트는 ' ," 이 둘다 문자열 표시로 사용한다.
다만 한번 시작되면 끝날때도 같은 따옴표를 써야한다.
예를 들어 '안녕", "안녕'와 같은 형식은 문법 오류를 불러온다.

"  "같이 쌍따옴표 안에는 '을 사용가능하다.
'   '같이 따옴표 안에는 "을 사용할수있다.
만약 "  "안에 쌍따옴표를 쓰고자 한다면 \" 같이 이스케이프 문자를 활용한다.
