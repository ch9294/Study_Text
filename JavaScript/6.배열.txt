배열

 1) 생성 방법

var alphabet = ['a','b','c'];
대괄호를 사용한다.

 2) 접근 방법
다른 프로그래밍 언어와 동일하게 인덱스를 사용하여 접근함/

 3) 제어 함수

push('d'....) : 배열의 끝에 매개변수로 입력한 값을 추가한다. 기존 배열에 원소를 추가하는 개념

concat('e'...) : 배열의 끝에 매개변수로 입력한 값을 추가한 새로운 배열을 만들어 반환함.
                 기존 배열에 추가하는 것이 아닌 새로운 배열을 만듬.

shift() : 첫번째 요소를 삭제

pop() : 마지막 요소를 삭제

sort() : 정렬한다.

reverse() : 뒤집는다.
